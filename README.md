# Github User Finder
Developed a responsive interface using Bootstrap 4 to fetch the profile of a GitHub user through the GitHub RESTful API

- [x] Bootstrap 4
- [x] ES6 Classes
- [x] Fetch
- [x] HTTP Methods
- [x] Promises
- [x] Async/Await
- [x] RESTful API
- [x] Responsiveness
